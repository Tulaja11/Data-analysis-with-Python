# Data-analysis-with-Python
This repository contains small data analysis projects created using Python.
The goal is to explore data, find patterns, and understand insights in an easy and practical way.

# Projects

## Hidden Patterns in Purchase Paths
- This project looks at customer purchase data to understand how people move from one step to another before buying something.

## Pattern Breaker Analytics
- This project focuses on finding unusual or unexpected patterns in data.

## Hospital Appointment Attendance & No-Show Pattern Analysis
- This project analyzes hospital appointment data to understand patterns in missed and cancelled appointments based on time slots, reasons for visit, and doctor-wise trends to improve scheduling efficiency and healthcare service delivery.

## Student Placement Trend & Eligibility Analyzer
- This project analyzes student academic and placement data to identify placement trends based on gender, specialization, and work experience, and applies rule-based eligibility criteria to determine placement readiness and support training and placement decision-making

# Tools Used

## Python
- Pandas
- NumPy
- Matplotlib

- Practice data analysis using Python
- Learn how to work with real data
