# Data-analysis-with-Python
This repository contains small data analysis projects created using Python.
The goal is to explore data, find patterns, and understand insights in an easy and practical way.

# Projects

##Hidden Patterns in Purchase Paths
- This project looks at customer purchase data to understand how people move from one step to another before buying something.

## Pattern Breaker Analytics
- This project focuses on finding unusual or unexpected patterns in data.

# Tools Used

## Python
- Pandas
- NumPy
- Matplotlib 

## Purpose
- Practice data analysis using Python
- Learn how to work with real data
- Improve problem-solving and analytical thinking
